## 📱 页面流程

### 完整用户流程

```
1. 访问首页
   ↓
2. 选择餐厅（自动定位，显示附近）
   ↓
3. 选择套餐（预算筛选，折扣展示）
   ↓
4. 确认订单（摘要展示，卡匹配）
   ↓
5. 下单成功（取餐码展示）
   ↓
6. 复制取餐码 / 再来一单
```

### 状态管理
```typescript
interface State {
  currentStep: 'store' | 'combo' | 'summary' | 'success';
  selectedStore: Store | null;
  selectedCombo: Combo | null;
  orderId: string | null;
  loading: boolean;
}
```
